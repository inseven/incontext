# Features

## Admonitions

InContext supports GitHub-style admonitions.

### Note

#### Markdown

```plaintext
> [!NOTE]
>
> This is a note.
```

#### Output

> [!NOTE]
>
> This is a note.

### Tip

#### Markdown

```plaintext
> [!TIP]
>
> This is a tip.
```

#### Output

> [!TIP]
>
> This is a tip.

### Important

#### Markdown
